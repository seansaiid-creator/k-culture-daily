# 🎌 K-Culture Daily

**K-POP Hidden Gems, K-Beauty, K-Snack 자동 큐레이션 매거진**

- 🌍 글로벌 타겟 (영어 기반)
- 🤖 완전 자동화 (손이 거의 안 감)
- 💰 월 $5-10 운영비
- ⚡ 매일 자동 업데이트
- 📊 실시간 분석

## 기술 스택

- **프론트엔드**: Next.js 14 + React + Tailwind CSS
- **백엔드**: Node.js + Express + MongoDB
- **자동화**: GitHub Actions + 24개 크롤러
- **배포**: Vercel + Railway
